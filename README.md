# Demo of Rails with server-side rendered React

_Currently work in progress. See [Issues](https://github.com/railslove/rails-ssr-react-demo/issues)._

## Setup

- ruby `2.6.4`
- node `>= 13.0.0`

```console
$ git clone
$ bundle install
$ yarn install
$ ./bin/webpack-dev-server
$ rails s
```
